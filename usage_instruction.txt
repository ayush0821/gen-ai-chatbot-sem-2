Usage Instructions


### 1. Create Virtual Environment

```
python3 -m venv venv

```

### 3. Install Dependencies

```bash

pip install streamlit openai python-dotenv
pip install groq

```

### 4. Setup API Key

Create a `.env` file and add your API key:

```
OPENAI_API_KEY=your_api_key_here
```

### 5. Run the Application

```
streamlit run app.py
```


### 7. How to Use

* Enter your query (e.g., "What is the fees?")
* The chatbot will generate a response using AI.
* You can ask multiple questions related to college information.
