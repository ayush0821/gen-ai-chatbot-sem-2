Usage Instructions


### 1. Create Virtual Environment

```
python3 -m venv venv

```

### 2. Install Dependencies

```bash

pip install streamlit openai python-dotenv
pip install groq

```

### 4. Run the Application

```
streamlit run app.py
```

* You can ask multiple questions related to college information.
